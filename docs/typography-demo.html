The Coven remembers.

A three-face system for the grimoire — one for identity, one for reading, one for inspecting.

This is Inter. It carries every paragraph, form field, control, table cell, chat bubble, docs page, and inline prose in the Coven. It's the sans you read.

Body copy should feel calm and dense enough to hold real information but never austere. Inter's optical adjustments make it work at 11px labels and 20px lead paragraphs alike.

Display / brand — EB Garamond

Headlines, hero copy, ceremonial spreads. Never body. Its italics carry the charm — use them for pull-quotes, kickers, and named entities in lore copy.

Rules

  • Serif is identity only — headlines, hero, ceremonial. Never body, never labels, never dense UI.
  • Sans is everything you read — body, chat, prose, controls, tables, docs, forms.
  • Mono is everything you inspect — code, terminal, identifiers, uppercase labels, badges, metadata.
  • Real italics matter — EB Garamond has a distinctive italic. Do not synthesize italics on fonts that don't ship one.
  • No playful fonts. No rounded display. No cursive. No novelty.
  • Weights: 400 body · 500 medium · 600 semi-bold section headers · 700 bold headlines.
  • Line heights: 1.4 sans body · 1.55 serif prose · 1.2 headings.
